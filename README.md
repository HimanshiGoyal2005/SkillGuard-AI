# SkillGuard AI

SkillGuard AI is an AI-powered system that verifies whether the skills mentioned in resumes are genuine.

## Features
- Resume parsing
- Skill extraction
- AI-generated tests
- Anti-cheating detection
- Skill authenticity score

## Tech Stack
Frontend: React  
Backend: Python FastAPI
AI Engine: Python (NLP + ML)
Database: PostgreSQL

## Project Goal
To build a system that detects fake skills in resumes by generating personalized tests and analyzing behaviour.
